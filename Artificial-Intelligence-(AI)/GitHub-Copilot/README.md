## Quick Orientation

- Most questions cover **GA (Generally Available)** features. Preview features appear only if commonly used.
- Exam may include **interactive components** (lab-style questions).
- Register with a **personal MSA account** — not a work/school AAD account (records lost if you leave org).
- Retake policy: wait **24 hours** after first failure; subsequent retakes vary.
- Practice sandbox: [GHCertDemo.starttest.com](https://GHCertDemo.starttest.com)
